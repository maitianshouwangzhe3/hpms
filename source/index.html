HPMS 项目介绍

项目概述

HPMS 是一个包含多种功能的项目，涉及核心代码、依赖库、示例代码等多个部分。

项目架构

以下是项目的架构图：

依赖库

  • Jemalloc: 高性能内存分配器
  • LuaJIT2: Lua 语言的即时编译器

示例代码

项目中包含多个示例代码，例如：

  • echo-cli.lua: 客户端示例
  • echo-srv.lua: 服务器示例
  • webserver.lua: Web 服务器示例

核心代码

核心代码位于 core/ 目录下，包含多个 C 文件和头文件，例如：

  • ae.c 和 ae.h: 事件循环相关代码
  • anet.c 和 anet.h: 网络相关代码
  • buffer.c 和 buffer.h: 缓冲区相关代码

测试代码

项目中包含多个测试代码，位于 test/ 目录下，例如：

  • test_connect_pool_cli.lua: 连接池客户端测试
  • test_connect_pool_srv.lua: 连接池服务器测试
